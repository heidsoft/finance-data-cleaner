## 账单模板格式说明

### 必须包含的列（自动识别）

系统会根据列名关键词自动匹配，无需完全匹配，支持以下列名变体：

| 数据项 | 支持的列名关键词 |
|--------|----------------|
| 订单金额/销售额 | `订单金额`、`商品总额`、`amount`、`total`、`gmv`、`sales` |
| 订单数 | `订单数`、`笔数`、`count`、`订单数量` |
| 佣金 | `佣金`、`commission`、`平台服务费`、`扣点` |
| 技术服务费 | `技术服务费`、`tech`、`服务费`、`平台费` |
| 补贴/返点 | `补贴`、`subsidy`、`奖励`、`返点`、`rebate` |
| 日期/账期 | `日期`、`账期`、`period`、`date`、`月份` |

> **提示**：列名中包含关键词即可，如"订单金额（元）"能匹配到"订单金额"

### 操作步骤

**Step 1：导出账单**
各平台后台 → 财务管理 → 账单/结算单 → 下载 Excel/CSV

**Step 2：导入账单**
1. 打开「账单对账」Tab
2. 点击「+ 导入账单」按钮
3. 选择对应平台的账单文件（支持多选，可同时导入多个平台）
4. 系统自动识别平台、解析账单金额、佣金、技术服务费、补贴

**Step 3：生成佣金计提表**
导入账单后，点击「重新生成」获取佣金自动计提表，可导出 Excel

| 平台 | 后台路径 | 导出位置 |
|------|---------|---------|
| 淘宝 | 商家中心 → 结算账单 | 佣金结算单/技术服务费账单 |
| 天猫 | 商家中心 → 账房 | 月度账单/佣金账单 |
| 京东 | 商家后台 → 结算管理 | 账单中心 → 佣金对账单 |
| 抖音 | 抖店后台 → 财务管理 | 账单中心 → 佣金账单 |
| 小红书 | 买手后台 → 数据中心 | 佣金结算单 |
