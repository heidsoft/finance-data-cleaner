# 数据清洗工具 - 技术方案

## 技术栈
- **Electron** + **React** + **TypeScript**
- **xlsx** - Excel/CSV 处理
- **Tailwind CSS** - UI 样式

## 核心功能

### 1. 数据导入
- 支持 CSV、Excel (.xlsx/.xls) 文件
- 拖拽上传 + 文件选择器
- 多文件同时导入

### 2. 数据清洗
- **去重** - 按指定列去重或全行去重
- **空行清理** - 删除空行/空列
- **格式规范化** - 去除前后空格、统一日期格式
- **列筛选** - 选择保留哪些列

### 3. 数据合并
- 多个文件纵向合并（按行拼接）
- 智能识别相同表头进行对齐

### 4. 数据导出
- 导出为 CSV 或 Excel
- 自定义输出文件名
