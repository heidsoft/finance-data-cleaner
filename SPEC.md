# 财务数据清洗工具 - 技术方案

## 技术栈
- **Electron** + **React** + **TypeScript**
- **xlsx** - Excel/CSV 处理
- **Tailwind CSS** - UI 样式

## 核心功能

### 1. 数据导入
- 支持 CSV、Excel (.xlsx/.xls) 文件
- 文件选择器导入
- 多文件同时导入

### 2. 数据清洗
- **去重** - 按指定列去重或全行去重
- **空行清理** - 删除空行/空列
- **格式规范化** - 去除前后空格、统一日期格式（YYYY-MM-DD）
- **列筛选** - 选择保留哪些列

### 3. 数据合并
- 多个文件纵向合并（按行拼接）

### 4. 数据导出
- 导出为 CSV 或 Excel

## 财务对账功能

### 账单对账
- 支持平台：淘宝、天猫、京东、抖音、抖音电商、快手电商、拼多多
- 自动识别平台（根据文件名）
- 自动解析：账单金额、佣金、技术服务费、补贴
- 生成佣金计提表
- **退款损失还原**：导入退款单，自动计算佣金损失（平台不退佣金）

### 品牌阶梯返利
- 输入月GMV，自动按阶梯计算返利金额
- 默认阶梯：0-50万×2%、50-100万×3%、100-200万×4%、200-500万×5%、500万以上×6%

### 月度汇总
- 按月、按平台汇总：GMV、订单数、佣金、技术服务费、补贴、净收款
- 支持月度环比对比
- 实际收入 = 净收款 - 预估佣金损失

### SKU映射
- 平台品名 → 内部编码 映射表
- 支持价格字段

## 数据模板

各平台账单模板位于 `templates/` 目录，参考 `templates/README.md` 了解详细格式要求。

### 支持的列名关键词

| 数据项 | 支持的列名 |
|--------|-----------|
| 订单金额 | `订单金额`、`商品总额`、`amount`、`total`、`gmv`、`sales` |
| 佣金 | `佣金`、`commission`、`平台服务费`、`扣点` |
| 技术服务费 | `技术服务费`、`tech`、`服务费`、`平台费` |
| 补贴/返点 | `补贴`、`subsidy`、`奖励`、`返点`、`rebate` |
| 日期 | `日期`、`账期`、`period`、`date`、`月份` |

## 项目结构

│   └── preload.ts      # 预加载脚本（contextBridge）
├── src/
│   ├── App.tsx         # 主组件（6个Tab全部逻辑）
│   ├── main.tsx        # React 入口
│   ├── components/
│   │   ├── DataTable.tsx       # 数据表格
│   │   ├── FileSidebar.tsx     # 文件列表
│   │   ├── MonthlySummary.tsx   # 月度汇总

│   ├── 淘宝账单模板.csv
